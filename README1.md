# AI-and-Swagger-Test-Automation
Swagger/OpenAPI specification: https://petstore.swagger.io/

Task:
Use the following Swagger/OpenAPI specification:
👉 https://petstore.swagger.io/
Use an AI tool of your choice (e.g. Copilot, agent, etc.) to:
- generate test cases based on the API documentation
- review and refine the generated test cases
- automate a selected subset of these test cases
- document your approach and findings briefly

Expected output
They should provide:
- selected endpoints from the Swagger Petstore API
- AI-generated test cases
- refined/validated test cases
- automated sample tests (working prototype)
- short summary of:
-- lessons learned
-- limitations of AI-generated results
-- where manual adjustments were required

Optional Guidance (for better results):
- Focus on key endpoints (e.g. Pet, Store, User) instead of full coverage
- Include positive, negative, and edge cases
- Keep automation simple but executable
- Highlight clearly AI vs. manual contribution
